✅ **Windows 7 یا بالاتر**

## مرحله 1: آماده‌سازی پایگاه داده

### گام 1: ایجاد فایل Access جدید

```
1. Microsoft Access را باز کنید
2. "فایل جدید" را کلیک کنید
3. یک نام بگذارید: Database.accdb
4. در یک فولدر شبکه‌ای ذخیره کنید
   (مثال: \\SERVER\shared\Database.accdb)
```

### گام 2: ایجاد جدول Users

```
1. در Access، "جدول جدید" را ایجاد کنید
2. فیلدهای زیر را اضافه کنید:

ستون              | نوع           | خصوصیات
-----------------|---------------|------------------
UserID            | AutoNumber    | Primary Key
Username          | Text (100)    | Unique
Password          | Text (255)    | 
FullName          | Text (255)    | 
Email             | Text (255)    | 
Role              | Text (50)     | 
IsActive          | Yes/No        | Default: True
CreatedDate       | DateTime      | 
CreatedBy         | Text (255)    | 
LastModifiedDate  | DateTime      | 
LastModifiedBy    | Text (255)    | 
LastLoginDate     | DateTime      | 
LoginAttempts     | Integer       | Default: 0

3. جدول را "Users" نام‌گذاری کنید
4. ذخیره کنید
```

### گام 3: ایجاد جدول Logs

```
1. جدول جدیدی ایجاد کنید

ستون              | نوع           
-----------------|---
LogID             | AutoNumber (PK)
UserID            | Long
Username          | Text (255)
ActionType        | Text (50)
Description       | Text (500)
OldValue          | Text (500)
NewValue          | Text (500)
LogDateTime       | DateTime
IPAddress         | Text (50)

2. جدول را "Logs" نام‌گذاری کنید
3. ذخیره کنید
```

### گام 4: ایجاد جدول AppSettings

```
1. جدول جدیدی ایجاد کنید

ستون              | نوع           
-----------------|---
SettingID         | AutoNumber (PK)
SettingName       | Text (100)
SettingValue      | Text (500)
IsEncrypted       | Yes/No
ModifiedBy        | Text (255)
ModifiedDate      | DateTime
Description       | Text (255)

2. جدول را "AppSettings" نام‌گذاری کنید
3. ذخیره کنید
```

### گام 5: تنظیم رمز عبور پایگاه داده

```
1. در Access، روی "فایل" کلیک کنید
2. "اطلاعات" → "رمز‌گذاری پایگاه داده" را انتخاب کنید
3. یک رمز عبور قوی تنظیم کنید (مثال: SecurePass123!)
4. رمز را یادداشت کنید (لازم خواهد شد)
```

### گام 6: ایجاد کاربر ادمین اولیه

```
1. جدول "Users" را باز کنید
2. یک رکورد جدید اضافه کنید:

UserID:           1 (خودکار)
Username:         admin
Password:         [از فایل VBA_MODULES.md کپی کردید]
FullName:         Administrator
Email:            admin@example.com
Role:             Admin
IsActive:         True
CreatedDate:      [تاریخ امروز]
CreatedBy:        System
LastModifiedDate: [تاریخ امروز]
LastModifiedBy:   System
LoginAttempts:    0

3. ذخیره کنید
```

## مرحله 2: آماده‌سازی اکسل

### گام 1: ایجاد فایل اکسل جدید

```
1. Microsoft Excel را باز کنید
2. یک فایل جدید ایجاد کنید
3. با نام MainProgram.xlsm ذخیره کنید
4. فرمت را "Excel Macro-Enabled Workbook" انتخاب کنید
```

### گام 2: ورود به VBA Editor

```
1. Excel را باز کنید
2. Alt + F11 را فشار دهید
3. VBA Editor باز می‌شود
```

### گام 3: اضافه کردن References

```
1. در VBA Editor، روی "Tools" کلیک کنید
2. "References" را انتخاب کنید
3. موارد زیر را چک کنید:
   - Visual Basic for Applications
   - Microsoft Excel Object Library
   - Microsoft Forms 2.0 Object Library
   - Windows Script Runtime
   
4. OK را کلیک کنید
```

### گام 4: اضافه کردن ماژول‌ها

```
در VBA Editor:
1. روی پروژه راست‌کلیک کنید
2. "Insert" → "Module" را انتخاب کنید
3. ماژول‌های زیر را ایجاد کنید:
   - ModEncryption
   - ModSettings
   - ModDatabase
   - ModLogging
   - ModUserManagement
   - ModUserProfile

(کدهای هر ماژول در فایل VBA_MODULES.md موجود است)
```

### گام 5: اضافه کردن فرم‌ها

```
در VBA Editor:
1. روی پروژه راست‌کلیک کنید
2. "Insert" → "UserForm" را انتخاب کنید
3. فرم‌های زیر را ایجاد کنید:
   - frmLogin
   - frmMain
   - frmSettings
   - frmCreateUser
   - frmEditUser
   - frmUserManagement
   - frmChangeOwnPassword
   - frmChangeUserPassword
   - frmLogs

(طراحی و کد هر فرم در فایل FORMS_CODE.md موجود است)
```

## مرحله 3: اتصال پایگاه داده

### گام 1: تنظیمات اولیه

```
1. فرم Settings را باز کنید
2. مسیر پایگاه داده را وارد کنید:
   \\SERVER\shared\Database.accdb
   
3. نام کاربری را وارد کنید:
   (نام کاربری Windows یا مشخص شده)

4. رمز عبور را وارد کنید:
   (رمزی که در Access تنظیم کردید)
```

### گام 2: تست اتصال

```
1. دکمه "Test Connection" را کلیک کنید
2. اگر اتصال موفق بود:
   ✓ "اتصال موفق" نمایش داده می‌شود
   
3. اگر خطا داشت:
   ✗ "خطا در اتصال" نمایش داده می‌شود
   (فایل TROUBLESHOOTING.md را بخوانید)
```

### گام 3: ذخیره تنظیمات

```
1. دکمه "Edit" را کلیک کنید
2. اطلاعات را تصحیح کنید (اگر لازم بود)
3. دکمه "Save" را کلیک کنید
4. تنظیمات ذخیره می‌شود
```

## مرحله 4: تست سیستم

### تست لاگین

```
1. برنامه را اجرا کنید
2. نام‌کاربری وارد کنید: admin
3. رمز عبور وارد کنید: 123456
4. دکمه "ورود" را کلیک کنید
```

### تست مدیریت کاربران

```
1. صفحه اصلی را باز کنید
2. "مدیریت کاربران" را کلیک کنید
3. دکمه "ایجاد کاربر جدید" را کلیک کنید
4. یک کاربر تست ایجاد کنید
5. لیست را بروزرسانی کنید
```

### تست تنظیمات

```
1. "تنظیمات" را کلیک کنید
2. دکمه "Edit" را کلیک کنید
3. دکمه "Test Connection" را کلیک کنید
4. نتیجه را مشاهده کنید
```

### تست لاگ‌ها

```
1. "مشاهده لاگ‌ها" را کلیک کنید
2. فعالیت‌های ثبت شده را مشاهده کنید
```

## استفاده از فایل‌های آماده

اگر نمی‌خواهید همه‌ی مراحل را دنبال کنید:

```
1. EXCEL_TEMPLATE.xlsm را دانلود کنید
2. DATABASE.accdb را دانلود کنید
3. هر دو فایل را در یک فولدر قرار دهید
4. اکسل را باز کنید
5. فرم Settings را باز کنید
6. مسیر پایگاه داده را تصحیح کنید (مسیر جدید)
7. اتصال را تست کنید
```
