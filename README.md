## 🎯 ویژگی‌های اصلی

### 1. سیستم لاگین
- ورود نام‌کاربری و رمز عبور
- تأیید اعتبار در پایگاه داده
- ثبت تاریخ و ساعت آخرین ورود
- محدود کردن تعداد تلاش‌های ناموفق

### 2. مدیریت کاربران
- **ایجاد کاربر جدید:**
  - نام‌کاربری، رمز عبور، نام کامل، ایمیل
  - انتخاب نقش (Admin/User)
  - تولید رمز عبور تصادفی

- **ویرایش کاربر:**
  - تغییر نام کامل، ایمیل، نقش
  - فعال/غیرفعال کردن

- **حذف کاربر:**
  - فقط توسط ادمین
  - ثبت در لاگ

- **جستجو و فیلتر:**
  - جستجو بر اساس نام، ایمیل، نام‌کاربری
  - فیلتر نقش و وضعیت
  - نمایش تعداد کاربران

### 3. تنظیمات
- مسیر پایگاه داده
- نام کاربری پایگاه داده
- رمز عبور پایگاه داده (رمزگذاری شده)
- تست اتصال
- بررسی وضعیت

### 4. لاگ و گزارش
- ثبت تمام فعالیت‌ها
- نام کاربر و تاریخ/ساعت
- نوع عمل
- مقادیم قدیمی و جدید
- IP Address کاربر

### 5. امنیت
- رمزگذاری رمز‌های عبور
- رمزگذاری تنظیمات حساس
- فقط ادمین برای تنظیمات
- محدودیت تعداد تلاش‌ها
- ثبت تمام فعالیت‌ها

## 📊 ساختار جداول

### جدول Users
```
UserID (AutoNumber) - شناسه منحصر
Username (Text) - نام‌کاربری (یکتا)
Password (Text) - رمز رمزگذاری شده
FullName (Text) - نام کامل
Email (Text) - ایمیل
Role (Text) - نقش (Admin/User)
IsActive (Yes/No) - فعال/غیرفعال
CreatedDate (DateTime) - تاریخ ایجاد
CreatedBy (Text) - ایجاد شده توسط
LastModifiedDate (DateTime) - آخرین ویرایش
LastModifiedBy (Text) - ویرایش شده توسط
LastLoginDate (DateTime) - آخرین ورود
LoginAttempts (Integer) - تعداد تلاش‌های ورود
```

### جدول Logs
```
LogID (AutoNumber) - شناسه منحصر
UserID (Long) - شناسه کاربر
Username (Text) - نام‌کاربری
ActionType (Text) - نوع عمل
Description (Text) - توضیحات
OldValue (Text) - مقدار قدیمی
NewValue (Text) - مقدار جدید
LogDateTime (DateTime) - تاریخ و ساعت
IPAddress (Text) - آدرس IP
```

### جدول AppSettings
```
SettingID (AutoNumber) - شناسه منحصر
SettingName (Text) - نام تنظیم
SettingValue (Text) - مقدار تنظیم
IsEncrypted (Yes/No) - رمزگذاری شده
ModifiedBy (Text) - ویرایش شده توسط
ModifiedDate (DateTime) - تاریخ ویرایش
Description (Text) - توضیحات
```

## 📝 دستورالعمل‌های اساسی

### لاگین
```
1. نام‌کاربری را وارد کنید
2. رمز عبور را وارد کنید
3. دکمه "ورود" را کلیک کنید
```

### صفحه اصلی
```
- مدیریت کاربران (فقط ادمین)
- تغییر رمز عبور (همه کاربران)
- تنظیمات (فقط ادمین)
- مشاهده لاگ‌ها (فقط ادمین)
- خروج
```

### مدیریت کاربران
```
ایجاد کاربر جدید:
1. دکمه "ایجاد کاربر جدید" را کلیک کنید
2. اطلاعات را وارد کنید
3. "ایجاد" را کلیک کنید

ویرایش کاربر:
1. کاربر را از لیست انتخاب کنید
2. دکمه "ویرایش" را کلیک کنید
3. اطلاعات را تغییر دهید
4. "ذخیره" را کلیک کنید

حذف کاربر:
1. کاربر را از لیست انتخاب کنید
2. دکمه "حذف" را کلیک کنید
3. تأیید کنید

فعال/غیرفعال کردن:
1. کاربر را از لیست انتخاب کنید
2. دکمه "فعال کردن" یا "غیرفعال کردن" را کلیک کنید
```

### جستجو و فیلتر
```
جستجو:
1. در فیلد "جستجو" بنویسید
2. نتایج خودکار فیلتر می‌شوند

فیلتر نقش:
1. نقش را از Combo انتخاب کنید

فیلتر وضعیت:
1. وضعیت را از Combo انتخاب کنید
```

## 🔐 نکات امنیتی

1. **رمز‌های عبور:** تمام رمز‌های عبور رمزگذاری شده‌اند (XOR + Base64)
2. **تنظیمات:** نام کاربری و رمز پایگاه داده رمزگذاری شده‌اند
3. **لاگ:** تمام فعالیت‌ها ثبت می‌شود
4. **مجوزها:** فقط ادمین می‌تواند تنظیمات را تغییر دهد
5. **رمز قوی:** حداقل 6 کاراکتر الزامی است
6. **IP Address:** IP Address کاربر ثبت می‌شود
